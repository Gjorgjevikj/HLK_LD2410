bool HLK_LD2410::disableEngineeringMode ( )

Sends the command to the sensor to exit engineering mode (goes back to basic target reporting mode) and waits for reply (ACK frame) from the sensor

If the sensor was already in command mode before calling this function it will remain in command mode
